Постановка задачи

Константа — это число, символ или символьная строка, которую можно использовать в программе в качестве значения. Константы служат для представления значений с плавающей запятой, целых чисел, перечислений или символов, изменение которых невозможно. Фактически это один из видов защиты в коде.

Строка - это последовательность ASCII или UNICODE символов. Строки в C, как и в большинстве языков программирования высокого уровня рассматриваются как отдельный тип, входящий в систему базовых типов языка. Так как язык C по своему происхождению является языком системного программирования, то строковый тип данных в C как таковой отсутствует, а в качестве строк в С используются обычные массивы символов.

Для объявления массива символов с строковой константой применяются ключевые слова const и char.

Формат записи:

const char идентифиткатор [число]= "возможная строка";

Примеры:

Массив символов с строковой константой с числом 0:

const char str[0]="";

Массив символов с строковой константой с числом:

const char str[6]="Hello"
